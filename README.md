# Neural Network & Deep Learning Lab Manual

This repository contains the official lab implementations for the 3rd-year B.Tech Computer Science & Engineering curriculum. The experiments cover the fundamental transition from basic mathematical operations to advanced Deep Learning architectures.

---

## 🔬 Lab Experiment Index

### Section 1: Foundations of Data Science
* **Experiment 1: Matrix Calculus with NumPy**
    * **Objective:** Implementing fundamental linear algebra operations including matrix transposition and axis-wise summation.
* **Experiment 2: Structured Data Handling via Pandas**
    * **Objective:** Construction and manipulation of DataFrames using dictionaries, lists, and zip functions.
* **Experiment 3: Numerical Optimization with SciPy**
    * **Objective:** Utilizing the `scipy.optimize` library to solve non-linear mathematical equations.
* **Experiment 4: Statistical Visualization**
    * **Objective:** Developing graphical representations of data using Matplotlib bar charts.

### Section 2: Perceptrons and Logic Gates
* **Experiment 5: Linear Perceptron Model**
    * **Objective:** Implementation of the Perceptron Learning Algorithm to simulate AND and OR logic gates.
    * **Focus:** Understanding weight updates and bias adjustments in a single-layer model.

### Section 3: Artificial Neural Networks (ANN)
* **Experiment 6: Multi-Layer Perceptron (XOR Problem)**
    * **Objective:** Solving non-linear separation using a hidden layer and manual backpropagation.
* **Experiment 7: Analysis of Activation Functions**
    * **Objective:** Implementation and visualization of Sigmoid, ReLU, Tanh, and Softmax functions.
* **Experiment 8: Full Neural Network Training Loop**
    * **Objective:** Designing a complete forward and backward propagation cycle with randomized weight initialization.

### Section 4: Advanced Optimization & RNNs
* **Experiment 9: Adam Optimizer Implementation**
    * **Objective:** Step-by-step coding of the Adam (Adaptive Moment Estimation) optimization algorithm from scratch.
* **Experiment 10: Recurrent Neural Network (RNN) Architecture**
    * **Objective:** Building a Sequential RNN model using the TensorFlow/Keras framework for time-series data.

## 🛠️ Technical Environment
* **Language:** Python 3.8+
* **Libraries Used:** * `NumPy`: For vectorized mathematical operations.
    * `Pandas`: For data structuring and analysis.
    * `Matplotlib`: For activation function and data visualization.
    * `TensorFlow/Keras`: For high-level neural network modeling.

---

## 📌 Implementation Highlights
* **Vectorization:** Most experiments utilize `numpy.dot` for efficient matrix multiplications instead of nested loops.
* **Non-Linearity:** Heavy focus on the Sigmoid function and its derivative for gradient-based learning.
* **Optimization:** Comparison between basic delta-rule updates and modern adaptive optimizers like Adam.
